Cadastro de Clientes - Passo a Passo

Caminho no sistema: Menu > Cadastros > Clientes > Novo Cliente

Dados obrigatórios:

  • Nome completo ou razão social
  • CPF (pessoa física) ou CNPJ (pessoa jurídica)
  • Telefone para contato
  • Endereço completo (logradouro, número, bairro, cidade)

Dados complementares importantes:

  • E-mail para contato
  • Data de nascimento (para pessoa física)
  • Indicador de IE (inscrição estadual)
  • Configuração de senha de acesso (6 dígitos)

Observação: Após o cadastro, o sistema gera automaticamente uma senha de 6 dígitos que pode ser visualizada na Ficha do Cliente (F8 ou ALT+1).

Incluir Dependentes - Passo a Passo

Caminho no sistema: Menu > Cadastros > Clientes > [Cliente] > Guia Dependentes

Dados necessários para cada dependente:

  • Nome completo do dependente
  • Grau de parentesco
  • Telefone de contato
  • Data de nascimento
  • Situação (ativo/inativo)

Controle por senha/biometria:

  • Gerar Senha Padrão: Sistema cria automaticamente senha de 6 dígitos
  • Cadastrar biometria: É possível cadastrar até 5 digitais por dependente
  • A senha fica visível na Ficha do Cliente (F8 ou ALT+1)

Controle de Acesso - Configuração

Para clientes e dependentes:

  • O sistema gera automaticamente uma senha de 6 dígitos
  • É possível cadastrar biometria (até 5 digitais)
  • A senha pode ser visualizada na Ficha do Cliente (F8 ou ALT+1)

Localização das configurações:

  • Para clientes: Na tela de cadastro do cliente
  • Para dependentes: Na Guia Dependentes do cliente

Importante: A senha padrão deve ser alterada no primeiro acesso pelo cliente/dependente.

Cadastro de Bairros - Passo a Passo

Caminho no sistema: Menu > Cadastros > Bairros > Novo Bairro

Dados necessários:

  • Nome do bairro
  • Cidade de referência
  • Taxa de entrega (se aplicável)
  • Zona de entrega
  • Entregador padrão

Configurações importantes:

  • Definir valor de taxa de entrega específica para o bairro
  • Vincular a uma zona de entrega existente
  • Associar um entregador padrão para o bairro

Comissão de Entregadores - Configuração

Caminho no sistema: Menu > Cadastros > Entregadores > [Entregador] > Aba Comissões

Tipos de comissão disponíveis:

  • Comissão por porcentagem sobre o valor da venda
  • Comissão por valor fixo por entrega
  • Comissão por faixa de valor

Configuração passo a passo:

  • Selecione o tipo de comissão desejado
  • Defina o valor ou porcentagem
  • Configure as regras específicas (horários, bairros, etc.)
  • Salve as configurações

Inativação de Dependentes - Procedimento

Como inativar: Acesse a Guia Dependentes do cliente e altere a situação para "Inativo"

Efeitos da inativação:

  • O dependente perde o acesso ao sistema
  • As autorizações em convênios são removidas automaticamente
  • O histórico de movimentações é mantido

Importante: Ao inativar um dependente, o sistema remove automaticamente todas as autorizações em convênios vinculadas a ele.

Observação: É possível reativar o dependente posteriormente, mas as autorizações precisarão ser reconfiguradas manualmente.
